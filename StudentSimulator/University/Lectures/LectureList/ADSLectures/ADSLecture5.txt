==========Лекція №5 - Insertation Sort==========

Insertion Sort — це алгоритм сортування,
який працює так само, як сортування карт у руках.

Ми беремо нову карту і вставляємо її
у правильне місце серед вже відсортованих.

Алгоритм дуже добре працює,
коли масив вже майже відсортований.

Складність:

Кращий випадок: O(n)
Гірший: O(n²)

дея алгоритму:

Масив умовно ділиться на дві частини:

Ліва частина — вже відсортована
Права частина — ще ні

Кожен новий елемент з правої частини
ми вставляємо в правильне місце ліворуч.

Приклад:

Масив: [5, 2, 4, 3]

Беремо другий елемент (2)
і вставляємо його в ліву частину:

[2, 5, 4, 3]

Беремо 4:
4 < 5 → зсуваємо 5 вправо
Вставляємо 4:

[2, 4, 5, 3]

Беремо 3:
3 < 5 → зсуваємо
3 < 4 → зсуваємо
Вставляємо 3:

[2, 3, 4, 5] 🎉

Чому алгоритм працює швидко на майже
відсортованих масивах?

Бо елементи майже не потрібно зсувати.

Якщо масив вже відсортований —
алгоритм просто один раз проходить його.

Детальніше:

Починаємо з другого елемента,
бо перший вже вважається відсортованим.

for i = 1 → n-1

Зберігаємо поточний елемент:

key = arr[i]

Тепер рухаємось вліво по відсортованій частині
і зсуваємо більші елементи вправо.

while j >= 0 && arr[j] > key
 arr[j+1] = arr[j]
 j--

Коли знайшли місце — вставляємо key:

arr[j+1] = key
