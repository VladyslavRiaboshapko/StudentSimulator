==========Лекція №6 - Класи та Об'єкти (ООП)==========

Клас — це креслення. Об'єкт — це предмет, збудований за цим кресленням.

Ідея:

У тебе є клас "Student". У нього є поля (ім'я, енергія) та методи (вчитися, спати).

Приклад:

class Student
{
    public string Name;
    public int Energy = 100;

    public void Study() { Energy -= 10; }
}

// Створюємо екземпляр:
Student vlad = new Student();
vlad.Name = "Влад";
vlad.Study();

Детальніше:

Об'єктно-орієнтоване програмування (ООП) дозволяє моделювати 
реальний світ у коді. Кожна сутність у твоїй грі має бути класом.
