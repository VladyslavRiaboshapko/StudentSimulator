==========Лекція №8 - Успадкування==========

Успадкування дозволяє створювати нові класи на основі вже існуючих.

Ідея:

Є "Батько" (загальний) і "Дитина" (спеціалізована).

Приклад:

class NPC { public string name; }

class Merchant : NPC // Купець успадковує NPC
{
    public string name;
    public int gold;
    public void Trade() { /* логіка торгівлі */ }
}

Детальніше:

Купець автоматично має поле `name`, бо він — NPC. 
Це дозволяє уникати дублювання коду.
