==========Лекція №7 - Конструктори==========

Конструктор — це спеціальний метод, який викликається 
автоматично в момент створення об'єкта (`new`).

Навіщо це:

Щоб одразу задати об'єкту початкові значення.

Приклад:

class Item
{
    public string title;
    public int price;

    public Item(string name, int cost)
    {
        title = name;
        price = cost;
    }
}

Item sword = new Item("Сталевий меч", 150);

Детальніше:

Конструктор завжди має таке ж ім'я, як і клас, 
і не має типу повернення (навіть void).
Це зручно для швидкого налаштування нових предметів в інвентарі.
