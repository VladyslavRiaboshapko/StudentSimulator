==========Лекція №4 - Масиви та Списки==========

Це структури для зберігання групи однотипних елементів (наприклад, інвентар).

Різниця:

1. Масив (Array) — має фіксований розмір.
2. Список (List) — може розширюватись (динамічний).

Приклад:

// Масив на 3 елементи
string[] weapons = { "Ніж", "Пістолет", "Автомат" };

// Список (потребує using System.Collections.Generic)
List<string> inventory = new List<string>();
inventory.Add("Яблуко");
inventory.Remove("Яблуко");

Детальніше:

Нумерація елементів завжди починається з НУЛЯ.
Перший елемент — це index [0]. Останній — [Length - 1].
