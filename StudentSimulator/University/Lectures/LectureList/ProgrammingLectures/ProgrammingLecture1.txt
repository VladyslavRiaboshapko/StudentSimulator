==========Лекція №1 - Змінні та Типи даних==========

Змінні — це "коробки" для зберігання інформації в пам'яті комп'ютера.

Основні типи в C#:

1. int — цілі числа (кількість монет, рівень).
2. float — дробові числа (HP, швидкість). Додаємо суфікс 'f'.
3. string — текст (ім'я гравця).
4. bool — так або ні (true/false).

Приклад:

int health = 100;
float speed = 5.5f;
string playerName = "Vladyslav";
bool isGameOver = false;

Детальніше:

Пам'ятай про статичну типізацію: ти не можеш покласти текст у змінну для чисел.
