==========Лекція №2 - Умовні конструкції (If/Else)==========

Умови дозволяють програмі приймати рішення: робити щось чи ні.

Логіка:

Якщо (умова істинна) -> виконуємо код.
Інакше -> виконуємо інший код.

Приклад:

if (money >= 500)
{
    money -= 500;
    Console.WriteLine("Предмет куплено!");
}
else
{
    Console.WriteLine("Недостатньо грошей.");
}

Детальніше:

Можна додавати `else if` для перевірки кількох умов поспіль. 
Використовуй оператори порівняння: 
== (дорівнює), != (не дорівнює), > , < , >= , <=.
