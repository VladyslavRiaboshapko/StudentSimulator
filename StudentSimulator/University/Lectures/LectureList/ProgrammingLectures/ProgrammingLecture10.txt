==========Лекція №10 - Простори імен (Namespaces)==========

Namespaces — це спосіб групування схожих класів у великих проектах.

Як працює:

Щоб використати класи з певного простору імен, ми пишемо `using`.

Приклад:

using System; // Дозволяє писати Console.WriteLine
using System.Collections.Generic; // Дозволяє використовувати List

namespace MyAwesomeGame
{
    class GameController { /* ... */ }
}

Детальніше:

Це як папки на комп'ютері. Без них тисячі класів перемішалися б, 
і виникли б конфлікти імен (наприклад, два різних класи з назвою "Player").
